🎯 HOW TO STARTUP (5-10 MINUTES)
═══════════════════════════════════════════════════════════════════════════

Step 1: Prepare Model
─────────────────────
    mkdir -p backend/models
    cp road_model.pth backend/models/

Step 2: Configure
─────────────────
    cd backend
    cat > .env << 'ENVEOF'
    SUPABASE_URL=https://your-project.supabase.co
    SUPABASE_ANON_KEY=your_key_here
    SUPABASE_SERVICE_ROLE_KEY=your_key_here
    CORS_ORIGINS=["http://localhost:3000"]
    ENVEOF

Step 3: Setup (Automated)
────────────────────────
    bash setup.sh

    OR manually:
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt

Step 4: Database
────────────────
    1. Open: https://supabase.com/dashboard
    2. SQL Editor → New Query
    3. Paste: migrations/01_create_predictions_table.sql
    4. Click: Run

Step 5: Start Backend
─────────────────────
    cd backend
    source venv/bin/activate
    uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

Step 6: Verify
──────────────
    Open: http://localhost:8000/docs
    Run: python test_prediction_api.py image.jpg

📊 ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════

Image Upload
    ↓
Validate (type, size)
    ↓
Load PyTorch Model (road_model.pth)
    ↓
Preprocess Image (224x224, normalize)
    ↓
Run Inference
    ↓
Get Prediction + Confidence + Probabilities
    ↓
Save to Supabase
    ↓
Return JSON Result

🚀 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

POST /predictions/predict
├─ Input: Image file (jpg/png, max 10MB)
├─ Auth: Bearer token (required)
└─ Output: {prediction, confidence, probabilities, id, ...}

GET /predictions/history
├─ Auth: Bearer token (required)
├─ Query: limit=10, offset=0 (optional)
└─ Output: [{predictions}...]
